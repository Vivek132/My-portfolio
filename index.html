My projects

Lets have a look on my Projects.

Driver Assistance System For Obstacle Detection Using CAN Protocol

1. The Driver Assistance System for Obstacle Detection using the Controller Area Network (CAN) protocol is an advanced technology that helps drivers avoid collisions with objects while driving.
2. The system uses this protocol to transmit data between the sensors and other devices in real-time.
3. The sensors are mounted on the front, rear, and sides of the vehicle to provide a comprehensive view of the surroundings.
4. The data from the sensors is then processed by a central control unit, which uses complex algorithms to determine the position of the obstacles

RTC Based Smart Plant Watering System

1. Developed an RTC-based smart plant watering system using I2C protocol, moisture sensor, RTC, and motor.
2. Designed logic to automate plant watering based on the moisture level in the soil and current time from the RTC module.
3. Conducted sensor calibration and implemented data acquisition to ensure accurate moisture measurements.
4. Developed firmware using microcontroller board Arduino and integrated sensors to create a functional embedded system. Employed circuit design principles to create a reliable and efficient design.
